Memory Skill v1.0
====================
记忆 + 推理 + 关联 + 感知增强 四位一体
11项自检+7项修复 | SimHash语义匹配 | 原生自动建边

首次使用步骤：
1. 执行 /memory session（自动初始化 + 加载记忆）
2. 执行 /memory 架构（确认模板就绪）
3. 之后所有新会话自动继承，无需重复操作

核心命令速查：
  /memory <你想做的事>      万能入口（自动识别意图）
  /memory session           会话初始化 + 记忆加载
  /memory 自检              11项全量检查 + 7项自动修复
  /memory 仪表盘            Token 统计

说明：
  · Memory = 知识图谱记忆引擎 + 推理引擎 + 感知增强
  · 无鉴权版本，开箱即用
  · 数据存储在 skill 目录下的 data/ 中
