# Memory — 你说人话，它自动搞定

> **v1.0 · 记忆引擎 · 安装即用 · 原生自动建边 · SimHash语义匹配 · 体裁感知检索**

---

## 30 秒上手

安装后，首次使用执行初始化：

```
/mem session
```

之后即可正常使用所有功能：

```
/mem 帮我记住：公司年会定在 12 月 25 号，地点国际会议中心 3 楼
/mem 年会什么时候在哪里？
/mem 分析这段数据：[粘贴数据]
/mem 总结这篇文章：[粘贴文本]
/mem 追溯 年会   ← 按需回溯知识来源
```

## 一句搞定所有事

### 主入口（覆盖 90% 场景）
```
/mem <你想做的事>
```
Memory 自动识别意图：学习知识 / 查询记忆 / 搜索关联 / 分析数据 / 总结文章 / 纠错 / 推理推荐 / 追溯来源。

### 工具入口
```
/mem 仪表盘        → Token 消耗/节省统计
/mem 设置          → 查看配置、开启/关闭自动感知
/mem 工作区        → 查看当前工作区及知识分布
/mem 自检          → 执行11项自检+7项自动修复
/mem 架构          → 查看架构状态
/mem 自动建边      → 查看自动建边运行状态
```

### 记忆管理
```
/mem 修改 旧文本|新文本     → 修改已记错的知识
/mem 删除 <关键词>          → 删除一条知识
/mem 记忆列表 [页码]        → 分页浏览当前工作区的记忆
/mem 待确认                → 查看自动学习的知识
```

### Workspace 管理
```
/mem 工作区               → 查看当前工作区 + 各工作区知识分布
/mem 工作区 global        → 切换到全局工作区
/mem 工作区 ws_项目名     → 切换到指定工作区
/mem 全局搜索 <关键词>    → 跨所有工作区搜索知识
```

## 核心能力

### 记忆引擎 — 你说一次，它记一辈子
- **学完就记住** — 零 embedding，秒级查询，永久存储
- **图扩散搜索** — 从"产品需求"自动关联到"技术方案""测试用例""上线计划"
- **智能纠错** — "派森"→"Python"、"物连网"→"物联网"，越用越聪明
- **上下文消歧** — SimHash 语义匹配，换种说法也能找到
- **原生自动建边** — 学新知识后自动检测与已有节点的关联，消除孤立节点

### 自动更新
- **智能合并** — 学习新知识时，如果与已有条目高度相似，自动更新旧条目而非重复新增
- **变更追踪** — 每次自动更新记录 updated_at 和 update_source
- **零误更新** — 相似度不够不会触发更新

### 记忆隔离
- **项目级隔离** — 每个项目的知识自动归属对应 workspace
- **智能识别** — 从工作目录路径自动提取项目名
- **全局知识共享** — global workspace 的知识所有项目可见

### 推理引擎 — 数据自己会说话
- **数据分析** — 自动提取数字、趋势、对比、统计
- **文章总结** — 提炼核心结论、关键论据、数据支撑
- **纠缠关联** — 图扩散搜索发现隐藏关联

### 感知增强 — 对话中自动帮你
- **自动感知意图** — 不用你选功能，Memory 自己判断
- **全自动记忆** — 无需说"记住"，AI 自动提取有价值信息学习
- **上下文注入** — 检测到用户话题与已有知识相关时，自动注入知识
- **会话感知** — 每次新对话自动加载完整上下文

## 自检系统（11项检查+7项修复）

```
/mem 自检          → 执行完整自检
```

**11项检查：**

| # | 检查项 | 修复能力 |
|---|--------|----------|
| 1 | 数据库完整性 | 清理空/损坏节点，重建索引 |
| 2 | 注入一致性 | 重写 _inject.md |
| 3 | 对话历史可达 | 需AI验证 |
| 4 | Memory日志 | 检测日志目录/文件可读性 |
| 5 | 纠缠场一致性 | 清理断裂关联 |
| 6 | workspace隔离 | 检测归属异常 |
| 7 | 版本状态 | 检查版本号 |
| 8 | Token数据库 | 计数一致性检查 |
| 9 | 备份时效 | 超过30天告警 |
| 10 | 架构注入状态 | 检查模板完整性 |
| 11 | 自动建边运行状态 | 统计建边数量 |
